### Raid-5

RAID-5 구성은 각각의 디스크에 패리티(Parity) 정보를 분산하여 저장하는 방식이다. 패리티는 데이터의 오류를 검출하고, 특정 디스크에 장애가 발생했을 때 나머지 디스크의 데이터와 패리티를 조합하여 손실된 데이터를 복구하는 데 사용되는 중복 정보이다.

따라서 Parity 용량은 구성 디스크 1개의 용량과 같다.
계산식은 아래와 같다.
C = (N-1) \* D (C:가능용량, N:DISK 개수, D:하드 용량)

### 디스플레이 매니저

디스플레이 매니저는 X 서버를 구동하고 사용자가 로그인할 수 있도록 하는 프로그램이다. X 서버는 그래픽 사용자 인터페이스(GUI)를 제공하며, 디스플레이 매니저는 X 서버를 구동하고 사용자가 로그인할 수 있도록 하는 프로그램이다.

런 레벨 5로 부팅 시 로그인 창을 통해 사용자 인증을 수행한다.
