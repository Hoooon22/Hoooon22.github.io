## 리눅스 시스템의 이해

### $user와 $USER의 차이

$user: 사용자 정의 변수
$USER: 시스템 변수 (현재 로그인한 사용자 계정명)

### export

export는 변수를 환경 변수로 등록하는 명령어이다.

- export PATH=$PATH:/home/user1

| 구성 요소   | 의미 및 역할                                                                                               |
| ----------- | ---------------------------------------------------------------------------------------------------------- |
| export      | PATH라는 변수를 시스템 전체(자식 프로세스 포함)에서 쓸 수 있게 내보냄.                                     |
| PATH        | 명령어 실행 경로를 저장하는 환경 변수. ls나 cp를 실행할 때 시스템이 이 경로들을 검색하여 실행 파일을 찾음. |
| =           | 오른쪽의 값을 왼쪽 변수에 대입 (공백이 없어야 함에 주의).                                                  |
| $PATH       | 기존에 설정되어 있던 PATH 값을 불러옴 (기존 값을 유지하기 위해 필수).                                      |
| :           | 경로와 경로 사이를 구분하는 구분자 (윈도우의 ;와 같은 역할).                                               |
| /home/user1 | 새로 추가하고 싶은 디렉터리의 절대 경로.                                                                   |
