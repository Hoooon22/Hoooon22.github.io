## bash 명령어

- ' ' 혹은 ()$ 로 묶인 명령어는 명령 대체가 가능
- () 기호를 이용하여 그룹 명령을 실행한 후에는 원래 상태로 전환된다. (ex) (cd /tmp; ls) -> /tmp로 이동했다가 원래 상태로 돌아옴
- more: 화면 단위로 출력
- xargs: 표준 입력으로 받은 문자열을 인자로 받아 명령어를 실행
  - echo a.txt | xargs ls -l -> ls -l a.txt 와 동일한 결과를 출력
- `` : 명령 대체 문자로서 이 안에 있는 문자열은 명령어로 인식
- ^문자열 : 문자열로 시작하는 것을 의미
- 문자열$ : 문자열로 끝나는 것을 의미

- abc = 123 : 쉘 변수
- export: 현재 쉘에 정의되어 있는 변수를 환경 변수로 등록
