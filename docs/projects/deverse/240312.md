## 개요
- 웹서버를 도커에 배포한다
- Host OS: Ubuntu 22.04.4 LTS

## Springboot 프로젝트 탑재
1. Dockerfile 파일 생성
2. Gradle Clean
3. build.gradle에 아래 코드 작성 후, 업데이트
    ```gradle
    jar {
        enabled = false
    }
    ```
4. Gradle bootJar 실행
5. docker build -t 실행하여 이미지 생성

- 현재 이미지 생성까지 하였다. 이를 로컬서버에 구동시켜야한다.
