## Command Stack v1.0.10 릴리즈: 구글 캘린더 양방향 동기화 및 UI 개선

이번 v1.0.10 업데이트에서는 많은 사용자분들이 기다리셨던 **구글 캘린더 양방향 동기화** 기능이 추가되었습니다. 이제 Command Stack의 강력한 작업 관리 기능을 구글 캘린더와 연동하여 더욱 효율적으로 일정을 관리할 수 있습니다.

### 1. 구글 캘린더 양방향 동기화 (Google Calendar Bidirectional Sync)

가장 큰 변화는 구글 캘린더와의 완벽한 연동입니다. 이제 Command Stack과 구글 캘린더 간의 데이터가 실시간으로 동기화됩니다.

#### Google → Command Stack

- **자동 동기화**: 앱 실행 시(세션 당 1회) 자동으로 구글 캘린더의 이벤트를 가져옵니다.
- **수동 동기화**: Calendar Board 헤더에 추가된 "Sync" 버튼을 클릭하여 언제든지 최신 상태로 업데이트할 수 있습니다.
- **Context 통합**: 구글 캘린더에서 가져온 이벤트는 자동으로 'Google' Context의 Task로 변환되어 관리됩니다.

#### Command Stack → Google

- **이벤트 생성**: Task 생성 시 "Sync to Google Calendar" 옵션을 체크하면 구글 캘린더에도 동시에 이벤트가 생성됩니다. ('Google' Context 선택 시 기본값으로 체크됨)
- **실시간 반영**: 연동된 Task를 수정하거나 삭제하면, 구글 캘린더에도 즉시 반영되어 두 플랫폼의 상태를 항상 일치시킵니다.

### 2. 인증 및 세션 유지 (Authentication & Session Persistence)

사용자 경험을 개선하기 위해 로그인 및 세션 관리 기능을 강화했습니다.

- **14일 세션 유지**: 로그인 세션이 14일 동안 유지되도록 개선하여, 매번 번거롭게 로그인할 필요가 없어졌습니다.
- **로그아웃 기능**: 사용자 프로필 메뉴를 통해 명시적으로 로그아웃할 수 있는 기능을 추가했습니다.

### 3. UI/UX 개선 (UI/UX Enhancements)

사용 편의성을 높이기 위해 인터페이스를 직관적으로 개선했습니다.

- **헤더 레이아웃 변경**: 사용자 아바타와 로그인 버튼을 Calendar Board의 우측 상단으로 이동하여 접근성을 높이고, 화면 공간을 더욱 효율적으로 활용하도록 했습니다.
- **동기화 상태 피드백**: 동기화 작업 시 시각적인 피드백을 추가하여, 현재 상태를 명확하게 인지할 수 있도록 했습니다.

---

이번 업데이트를 통해 Command Stack이 단순한 로컬 작업 관리 도구를 넘어, 클라우드 캘린더와 유기적으로 연결되는 허브 역할을 하게 되었습니다. 앞으로도 개발자의 생산성을 극대화할 수 있는 다양한 기능들을 추가해 나갈 예정입니다.
