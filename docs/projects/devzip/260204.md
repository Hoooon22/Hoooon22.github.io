## Command Stack v1.0.11 릴리즈: Terminal-style Memo Mode 🎉

이번 v1.0.11 업데이트에서는 개발자들에게 익숙한 **터미널 스타일 메모 모드**가 새롭게 추가되었습니다! 마치 실제 터미널을 사용하는 듯한 경험으로 메모를 작성하고 관리할 수 있습니다.

### 1. Memo Mode 소개

터미널 명령어 스타일로 자유롭게 메모를 관리할 수 있는 새로운 모드입니다. vim 사용자라면 더욱 친숙하게 느껴질 것입니다.

#### 지원 명령어

| 명령어                         | 설명                                |
| ------------------------------ | ----------------------------------- |
| `vi <파일명>` / `vim <파일명>` | 메모 작성/수정 (ESC로 저장 후 종료) |
| `ls`                           | 저장된 메모 파일 목록 보기          |
| `cat <파일명>`                 | 메모 내용 보기                      |
| `rm <파일명>`                  | 메모 파일 삭제                      |
| `clear`                        | 화면 초기화                         |
| `help`                         | 도움말 보기                         |

### 2. 주요 변경사항

- **ViewSwitcher에 Memo Mode 버튼 추가**: 기존 모드들과 함께 간편하게 Memo Mode로 전환할 수 있습니다.
- **MemoBoard 컴포넌트 신규 생성**: 터미널 프롬프트 스타일의 인터페이스를 구현했습니다.
- **LocalStorage 기반 메모 저장**: 브라우저를 새로고침해도 메모가 유지됩니다.
- **ESC 키로 저장 및 종료**: vim 사용자에게 익숙한 조작 방식을 채택했습니다.

### 3. 개발자 친화적 UI/UX

이번 Memo Mode는 개발자들의 워크플로우를 고려하여 설계되었습니다:

- **터미널 프롬프트 스타일**: `user@commandstack:~$` 형태의 친숙한 프롬프트
- **명령어 기반 조작**: 마우스 없이 키보드만으로 모든 작업 가능
- **심플한 텍스트 기반 메모**: 빠른 아이디어 기록에 최적화

---

Command Stack은 이번 업데이트를 통해 작업 관리뿐만 아니라 빠른 메모 기능까지 제공하게 되었습니다. 개발 중 떠오르는 아이디어나 간단한 노트를 터미널 환경에서 바로 기록해보세요!
